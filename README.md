## Install

```
npm i palabra -g
```

# Usage

First thing you should do is:

- ✅ create `.palabra.json` with soft you want to install, like this:

```json
{
    "letras": {
        "fasm": "*"
    }
}
```

Then run:

```sh
palabra
```

That's it!
